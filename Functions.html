Functions

The following functions are available globally.

  • getVenues(_:)

    Retrieves a list of all venues your credentials have access to. It will fill in the name, logo, and cover image. Further details (like the maps and locations) will not be populated until youc all getVenue() with a given venue.

    Make sure you have set your MappedInUserName and MappedInSecret in your application’s info.plist file. If you don’t know what that is, please contact your MappedIn represenative.

    Parameter

    Parameter callback: A function that will be called with a list of venues retrieved when the operation is completed.

    Swift

  • getVenue(_:accessible:callback:)

    Retrives the full details (map, location, nodes, etc) of a given venue. If a specific venue had been loaded previously, that is cleared out to free memory.

    Parameter

    Parameter accessible: Specify whether you want accessible directions or not (ie, ignore stairs). False by default.

    Parameter

    Parameter callback: A function that will be called with the full Venue details when they have come back from the network
