MapViewCamera

public class MapViewCamera

The MapViewCamera allows you to control the camera on the MapView. Adjust the position, rotation, and zoom level

  • focusZoomScale

    The percentatge of the minimum zoom level to fit to a polygon we should zoom to

    Declaration

    Swift

    public var focusZoomScale:Float = 1.2
  • maxPitch

    This property controls the maximum camera pitch, in radians. If you set this too high, you would be able to rotate under the venue.

    Declaration

    Swift

    public var maxPitch: Float = 1.2
  • minPitch

    This property controls the minimum camera pitch, in radians. If you set this below 0, the user will be able to pitch so everything is upside down

    Declaration

    Swift

    public var minPitch: Float = 0.0
  • maxZoom

    This property controls how far the camera can zoom out (actually how far the camera is from the scene). In orthographic mode, it controls the maximum orthographic scale. The property will not change if you switch back and forth between perspective and orthographic mode.

    Declaration

    Swift

    public var maxZoom: Float = 10000
  • minZoom

    This property controls how far the camera can zoom in (actually how far the camera is from the scene). In orthographic mode, it controls the minimum orthographic scale. The property will not change if you switch back and forth between perspective and orthographic mode.

    Declaration

    Swift

    public var minZoom: Float = 10
  • animationDuration

    Length of time to animate from one camera position/orientation/zoom to another

    Declaration

    Swift

    public var animationDuration: Double = 0.3
  • usesOrthographicProjection

    Set true to switch to an orthographic projection (ie, no perspective). The zoom property will now control the orthograpic scale.

    Controls the camera’s x and y position. NOTE: The coordinates used don’t really mean anything at this stage, so don’t hardcode them as they will change.

    Controls the pitch (up and down) and yaw (clockwise around the map) of the camera. 0 pitch and 0 yaw would be looking straight down on the map in it’s default orientation
