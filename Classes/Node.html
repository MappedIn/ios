Node

public class Node: Hashable

A Node represents specific coordinates on a map, used for pathing. A path is just a sequential list of Nodes from one place to another.

Typically the start and end points of a path will belong to a specific location, but this is not actually required, and the Node itself has no knowledge of who may or may not own it.

A Node knows what map it belongs to, and it’s coordinates in 3D space (though the Z coordinate will be a constant across all nodes)
