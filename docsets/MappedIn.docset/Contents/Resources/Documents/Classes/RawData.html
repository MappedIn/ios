RawData

public class RawData

A wrapper class representing MappedIn JSON data. Provides a convient, robust, and type safe way for casting JSON data to concrete types.

    Creates a chainable key lookup. You can safely perform nested property lookups even if the property does not exist. Value unwrapping doesn’t occur until you try to cast to a value.
