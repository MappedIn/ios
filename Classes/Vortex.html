Vortex

public class Vortex: Location

A Vortex is a special entity that represents a link between two or more maps at a given venue. It is typically something like an elevator or stairs.

They can be marked (wheelchair) Accessible or not for pathfinding purposes. If the user has selected Accessible directions, paths that include non-accessible Vortexes like Stairs will not be used.
