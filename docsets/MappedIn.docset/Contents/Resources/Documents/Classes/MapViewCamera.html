MapViewCamera

public class MapViewCamera

The MapViewCamera allows you to control the camera on the MapView. Adjust the position, rotation, and zoom level

  • maxCameraPitch

    This property controls the maximum camera pitch, in radians. If you set this too high, you would be able to rotate under the venue.

    Declaration

    Swift

    public var maxCameraPitch: Float = 1.2
  • animationDuration

    Length of time to animate from one camera position/orientation/zoom to another

    Declaration

    Swift

    public var animationDuration: Double = 0.3
  • position

    Controls the camera’s x and y position. NOTE: The coordinates used don’t really mean anything at this stage, so don’t hardcode them as they will change.

    Declaration

    Swift

    public var position: (x: Float, y: Float)
  • rotation

    Controls the pitch (up and down) and yaw (clockwise around the map) of the camera. 0 pitch and 0 yaw would be looking straight down on the map in it’s default orientation

    Declaration

    Swift

    public var rotation: (pitch: Float, yaw: Float)
  • zoom

    Controls how close the camera is to the map. The units used here don’t mean anything right now

    Declaration

    Swift

    public var zoom: Float
  • focusOn(_:)

    Focuses the camera on a specific node. Will change maps if required. Doesn’t affect zoom level. Parameter node: The node to focus on

    Declaration

    Swift

    public func focusOn(node: Node)
  • focusOn(_:autoZoom:)

    Focuses the camera on a specific Polygon

    Parameter polygon: The Polygon to focus in on Parameter autoZoom: Whether the camera should automatically zoom in, out or both to fit the polygon into view. Can also specify none for no zoom behaviour.
