  • clearPaths(_:)

    Clears all paths currently drawn on the MapView. Also optionally forgets the current path so they won’t get redrawn later.

    Used automatically on map transitions to wipe the path before drawing any newly-visible nodes on the new map. You may also want to use it to temporarly clear the current path, or if the user is done navigating.

    Parameter

    Parameter forgetCurrentPath: Defaults to true. Forgets current path so it won’t be re-drawn later.

    Declaration

    Swift

    public func clearPaths(forgetCurrentPath: Bool = true)
