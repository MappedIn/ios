ARMapView

public class ARMapView: UIView

An ARMapView is similar to the MapView but instead shows a way to overlay the venue information onto the world. The mall’s geometry is hidden, but elements can be placed in the view just like a MapView to allow the user to see pathing or other points of interest.

    Sets the MapView to display a specific map associated with the current venue.

    Note: Any Elements that existed that were not map specific will be present after a new map is loaded.
