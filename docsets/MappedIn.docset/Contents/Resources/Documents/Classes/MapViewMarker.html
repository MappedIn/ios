MapViewMarker

public class MapViewMarker: Hashable

A visual element that can be placed overtop of a Mapview. Currently, only the child MapViewMarker2D can actually be used, with either an image or text.

    If you would like to do something when this marker is tapped set this property. The function will be executed with the Marker that was tapped, for your reference. If it’s not set, touches will pass through this marker to any underlying markers or polygons.
