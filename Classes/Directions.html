Directions

public class Directions

A Directions object will contain a list of Nodes on a path, and a corresponding list of Direction objects representing the instructions to display to the user.

  • path

    A list of nodes on a path. Pass this to your MapView’s .drawPath() to display it to the user

Instruction
  • Instruction

    An instruction for a user to get from one Node to another. It will have a Node it happens at, an Action (depart, arrive, turn, take vortex), and optional Locations to perform this at or towards, as well as a String representing the text instructions to display to the user.

    The at/towardsLocation properties are to let you mark or highlight key reference locations on the MapView if they will help the user find or keep their bearings.
