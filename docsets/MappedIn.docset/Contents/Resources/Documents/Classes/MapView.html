MapView

public class MapView: UIView, SCNSceneRendererDelegate

MapView is a custom view designed to handle displaying Maps for a Venue. Set the venue property and then call loadScene() to display the first map. From there, you can change maps (usually different floors) with the .setFloor() function` You can also highlight polygons and attach floating Markers. Since: 0.1 Author: MappedIn
