ImageCollection

public class ImageCollection

A collection of references to images of various sizes. Use myImageCollection[size] to get an image of that size or greater (in display units multiplied by display scale). Gives you the URL of the image you need to download.

  • subscript(_:)

    Tries to return an image url whose maximum dimension meets or exceeds the specified value in display units multiplied by display scale. If resolution is not available will return largest available.
